# MuscleECM_Project
This repository contains simulations and visualizations for modeling muscle stem cell–ECM fiber interactions in 3D, using PyQtGraph, OpenGL, and FIJI.

## Folder/Subfolder Overview:

### Videos_of_Visualizations:
Holds .mp4 recordings of key simulation outputs, capturing the behavior of integrins and ECM fibers over time. These are useful for presentation and analysis.

### FIJI Network Image Analysis:
Holds instructions on how to utilize FIJI software to skeletonize and convert SEM images of a cellular network into usable input files for the final simulation. In addition, it also contains a folder that shows how to utilize FIJI for pore size analysis of images to compare between young and old cellular networks.

### Visualizations:
Contains the subfolders 3D Final Visualization, Planar Visualizations, Cell, Fiber, and Old Visualizations.

### 3D Final Visualization:
Subfolder that contains the latest final full simulation visualization rendered in 3D using PyQtGraph. This simulation includes interactive and dynamic integrin activation, fiber deformation, spatial organization, and multiple cells being added.

### Planar Visualizations:
Subfolder that contains coding files demonstrating integrin catch slip bond mechanics and fiber fiber interactions in a planar environment.

### Cell:
Subfolder that contains a coding file that represents 1 single 3D cell modeled using OpenGL containing with integrins, clusters, and focal adhesions.

### Fiber:
Subfolder that contains a coding file that represents mechanics of an extracellular fiber that is deformable and elastic.

### Old Visualizations:
Subfolder that contains previous 2D and 3D visualizations created using matplotlib and OpenGL. These were used in earlier development stages for prototyping.

# MuscleECM_Project
This repository contains simulations and visualizations for modeling muscle stem cell–ECM fiber interactions in 3D, using PyQtGraph, OpenGL, and FIJI.

## Folder Overview:

### New Visualizations (OpenGL):
Contains the latest integrin-fiber interaction visualizations rendered in 3D using PyQtGraph. These include interactive and dynamic simulations of integrin activation, fiber deformation, spatial organization, and the final full simulation of the entire network and multiple cells being added to it.

### Old Visualizations (Matplotlib):
Archives previous 2D and static 3D visualizations created using matplotlib. These were used in earlier development stages for prototyping before switching to OpenGL.

### Videos_of_Visualizations:
Holds .mp4 recordings of key simulation outputs, capturing the behavior of integrins and ECM fibers over time. These are useful for presentation and analysis.

### FIJI Network Image Analysis:
Holds Instructions on how to utilize FIJI software to skeletonize and convert SEM images of a cellular network into usable input files for the final simulation. In addition, it also contains a folder that shows how to utilize FIJI for pore size analysis of images to compare between young and old cellular networks.
